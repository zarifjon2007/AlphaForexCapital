# Video fayllar

Agar `red-waves.mp4` o‘ynamasa, sababi: **Git LFS pointer** — fayl haqiqiy video emas, faqat havola (Loyiha ZIP yoki LFS siz clone qilingan bo‘lishi mumkin).

**Nima qilish kerak:**

1. **Haqiqiy videoni olish**
   - Asl repo’ni **git clone** qiling, keyin papkada: `git lfs install` va `git lfs pull` — shunda `public/videos/` dagi haqiqiy MP4 yuklanadi.
   - Yoki loyiha egasidan **red-waves.mp4** (taxminan 500 MB) ni so‘rab oling.

2. **Pointer’ni almashtirish**
   - Olingan yoki yuklab olgan **haqiqiy** `red-waves.mp4` faylini shu papkaga (`public/videos/`) qo‘ying va mavjud faylni ustiga yozing.

Shundan keyin Hero’dagi video brauzerda o‘ynashi kerak.
